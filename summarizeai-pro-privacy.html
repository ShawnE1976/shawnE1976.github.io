SummarizeAI Pro - Privacy Policy

Last updated: April 16, 2026

Overview

SummarizeAI Pro is a Chrome extension that summarizes webpages and YouTube videos using Claude AI. Your privacy is important to us. This policy explains what data the extension accesses and how it is used.

Data Collection

SummarizeAI Pro does not collect, store, or transmit any personal information. The extension does not track you, and does not use cookies or analytics.

Data Usage

When you click the summarize button, the extension reads the text content of the current webpage or YouTube video transcript. This content is sent directly to the Anthropic API (Claude AI) to generate a summary. The content is not stored by the extension after the summary is generated.

Data Stored Locally

  • API Key: Your Claude API key is stored locally in Chrome's storage on your device. It is never transmitted to any server other than the Anthropic API.
  • Subscription Status: Your payment status is stored locally via ExtensionPay.

Third-Party Services

  • Anthropic API: Page content is sent to Anthropic's API for summarization. See Anthropic's Privacy Policy.
  • ExtensionPay: Used for subscription management. See ExtensionPay's Privacy Policy.

Data Sharing

We do not sell, trade, or transfer your data to any third parties. Data is only sent to the Anthropic API for the sole purpose of generating summaries.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
