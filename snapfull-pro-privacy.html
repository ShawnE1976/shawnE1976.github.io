SnapFull Pro - Privacy Policy

Last updated: April 16, 2026

Overview

SnapFull Pro is a Chrome extension for full page screenshots with AI-powered image descriptions using Claude. Your privacy is important to us.

Data Collection

SnapFull Pro does not collect, store, or transmit any personal information. The extension does not track you, and does not use cookies or analytics.

Data Usage

When you take a screenshot and request an AI description, the screenshot image is sent to the Anthropic API (Claude AI) to generate a description. Screenshots are not stored by the extension after processing.

Data Stored Locally

  • API Key: Your Claude API key is stored locally in Chrome's storage on your device. It is never transmitted to any server other than the Anthropic API.
  • Subscription Status: Your payment status is stored locally via ExtensionPay.

Third-Party Services

  • Anthropic API: Screenshot data is sent to Anthropic's API for image descriptions. See Anthropic's Privacy Policy.
  • ExtensionPay: Used for subscription management. See ExtensionPay's Privacy Policy.

Data Sharing

We do not sell, trade, or transfer your data to any third parties. Data is only sent to the Anthropic API for the sole purpose of generating image descriptions.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated date.
